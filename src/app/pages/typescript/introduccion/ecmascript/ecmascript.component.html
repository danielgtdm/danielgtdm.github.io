¿Qué es ECMAScript?

ECMAScript es una estandarización para lenguajes de scripting, como JavaScript. Esta estandarización tiene como objetivo asegurar la interoperabilidad en la web, garantizando que varios navegadores web interpreten el código JavaScript de la misma manera y permitiendo a los desarrolladores crear un único software compatible con todos ellos.

Existen múltiples versiones de este estándar, y los navegadores web populares indican su compatibilidad con cada una de ellas.
